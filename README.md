# NLP_SPAM_SMS_DETECTION
Performed the following tasks in Python using the appropriate libraries, where applicable:
1. Loading of the data into a Pandas data frame.
2. Tokenization into words.
3. Normalization using stemming and lemmatization.
4. Stopword removal of common English words.
5. Part-of-speech tagging on the tokenized text.
6. Output the processed text, by combining processed text into a single string for each message.
and saving the text in a new column in the data frame.
